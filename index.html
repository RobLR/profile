About

My name is Robert Le Roux and I'm currently based in Leeds. I'm currently attending a Tech Talent Accelerator pre apprenticeship with the aim of getting an apprenticeship or junior developer role.

As a tech enthusiast I was drawn to computer science at GCSE level where I first experienced coding in Python and some basic HTML. After my A levels I started a Geology degree during which I chose modules involving statistical analysis with R and a module using GIS, which I used in further university projects.

I realised I wasn't enjoying Geology that much and decided to stop my course and started looking for ways into software engineering.
